Disable all your WordPress theme updates on plugin activation.

== Description ==

This plugin is use to disable all your theme updates as well as notifications for the same. It will prevent your wordpress to check for new updates for each theme. This plugin is ideal for whom who have customzied the theme and won't loose their customization at time of update. 

<strong>Read This First:</strong>
You need to keep your WordPress updated with themes and plugins so if you fail for the same then it will might compromise security of your site. 

== Installation ==

1. Extract the zip file into your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. On activation it will hide all the theme updates
